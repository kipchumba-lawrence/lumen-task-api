# Lumen Task Management API

## Description
A simple RESTful API built using Lumen for managing tasks. This API allows users to create, read, update, and delete tasks, with features for filtering, pagination, and searching.

## Installation

### Requirements
- PHP 7.4 or higher
- Composer
- PostgreSQL
